# Hangman - React + TypeScript + Vite + Redux + React-Router-DOM

## Running the game

1. clone to local computer
2. go to Root folder
   `cd game-hangman`
3. rename `.env.example` to `.env.local`
4. use package manager of choice (this is sample for pnpm):
   `pnpm install && pnpm dev`
5. for tests:
   `pnpm test`
